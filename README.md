# 🌐 template-next-16-intl

A modern Next.js 16.1.0 starter template with production-ready internationalization (i18n) setup. Built with next-intl, Tailwind CSS, and full RTL support for Arabic.

## ✨ Features

- **🚀 Next.js 16.1.0** - Latest features with Turbopack for fast development
- **🌍 Internationalization** - Full i18n support with next-intl
  - Support for English (en), German (de), and Arabic (ar)
  - RTL layout support for Arabic
  - Automatic locale detection and routing
- **🎨 Tailwind CSS** - Utility-first CSS framework with dark mode support
- **📱 Fully Responsive** - Mobile-first design approach
- **⚡ Static Site Generation** - Pre-rendered pages for all locales
- **🔄 Language Switcher** - Beautiful dropdown component for locale switching
- **📝 Complete Page Templates** - Home, About, and Contact pages with translations
- **🎯 TypeScript** - Type-safe development
- **🔧 Modern Tooling** - ESLint, PostCSS configured

## 📦 Pages Included

### 🏠 Home Page
- Hero section with CTA buttons
- Features grid with icons
- Statistics counter
- Testimonials section
- Call-to-action section

### 👥 About Page
- Mission & Vision cards
- Company values showcase
- Team section

### 📞 Contact Page
- Contact form with validation
- Business information cards
- Social media links

## 🛠️ Tech Stack

- **Framework**: Next.js 16.1.0
- **Internationalization**: next-intl ^4.8.3
- **Styling**: Tailwind CSS ^4
- **Language**: TypeScript ^5
- **Linting**: ESLint ^9
- **Fonts**: Geist, Geist Mono
